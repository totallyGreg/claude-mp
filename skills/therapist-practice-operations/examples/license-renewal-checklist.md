## Pre-Renewal Timeline

### 18 Months Before Expiration
- [ ] Confirm exact license expiration date (check license card and state website)
- [ ] Document deadline in calendar
- [ ] Note CE hour requirement for your state (example: 20 hours per 2-year cycle)
- [ ] Begin planning CE courses to complete

### 12 Months Before Expiration
- [ ] Start taking CE courses if not yet completed
- [ ] Create spreadsheet to track CE courses and hours
- [ ] Identify any required CE topics (example: ethics, diversity, domestic violence)
- [ ] Plan remaining courses to meet state requirement

### 6 Months Before Expiration
- [ ] Verify all CE courses are documented with certificates
- [ ] Review total CE hours completed vs. required hours
- [ ] Complete any remaining CE courses if needed
- [ ] Request transcripts or copies of course certificates from providers

### 3 Months Before Expiration
- [ ] Gather all CE certificates and supporting documents
- [ ] Organize documents in renewal folder (example: `WA_License_Renewal_2027/`)
- [ ] Review state renewal application form (download from state board website)
- [ ] Prepare any additional documents required (CYE form, liability insurance proof, etc.)

### 30 Days Before Expiration
- [ ] Submit online renewal application to state board
- [ ] Provide CE documentation as required
- [ ] Pay renewal fee
- [ ] Keep confirmation of submission
- [ ] Upload all supporting documents to state online portal

### At Expiration Date
- [ ] Monitor state board portal for renewal approval
- [ ] Expect approval within 2-3 weeks (varies by state)
- [ ] Print and save new license certificate
- [ ] Update license information in your practice management system
- [ ] Verify updated license appears on state board public lookup

## Important Notes

- **Always verify with your state board:** Requirements change. Confirm current requirements on official state board website before starting renewal process.

- **Start early:** Don't wait until the last minute. License renewal delays can interrupt your practice.

- **Keep old certificates:** Retain copies of old license and CE certificates for 7 years in case of audit or dispute.

- **Calendar reminders:** Set phone reminders at 12 months, 6 months, 3 months, and 30 days before expiration.

- **Multi-state tip:** Create state-specific calendars if licensed in multiple states; set separate reminders for each state.
