# GTD Coach

Pure GTD methodology coaching skill that provides David Allen's Getting Things Done principles, workflow guidance, and system health assessment. Works alongside omnifocus-manager for data-grounded coaching — this skill handles the "why" and "how" of GTD, while omnifocus-manager handles the "what" (actual OmniFocus queries and mutations).

## Capabilities

- Coach users on next action clarity, project definition, and capture/clarify/organize workflows
- Guide weekly reviews with GTD-grounded checklists
- Explain Horizons of Focus and their practical application
- Provide data-grounded coaching by integrating gtd-queries.js diagnostic results
